Buttons

A call-to-action (CTA) button is an interactive UI element both web and mobile.

Primary Button

The primary button is reserved for the most important call to action on the page.

Code

            
              <button href="#" class="btn-primary">Primary Button</button>

Secondary Button

The secondary button is reserved for actions of medium importance or when there are multiple calls to action that are equally important.

Code

            
              <button href="#" class="btn-secondary">Secondary Button</button>

Accessibility

All buttons should follow A11Y standards such as adequate color contrast, be legible and clearly labelled according to the action.
